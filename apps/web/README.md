# Web Frontend

Next.js frontend for Woolf monorepo.

## Development

```bash
pnpm dev
```

App will start on http://localhost:3000

## Build

```bash
pnpm build
```

## Production

```bash
pnpm start
```

## Pages

- `/` - Home page with API connection and database status

## Environment

Fetches API data from `http://localhost:3001`. Ensure the API server is running before starting the web app.
